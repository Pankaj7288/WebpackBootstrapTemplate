# Webpack Bootstrap Template

Webpack Bootstrap Empty project with Pankaj & Satu Samonty.

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running

```sh
npm run start
```

## Remove dist

```sh
npm run clean
```
